## 인코딩

  * ISO_8859_1(매개 변수)
  * US_ASC2(매개 변수)
  * UTF8(매개 변수)
  * UTF16(매개 변수)
  * UTF16BE(매개 변수)
  * UTF16LE(매개 변수)

## 매개 변수
  
  * 매개변수 -> 리턴타입
  * String -> byte[]
  * char -> byte
  * char[] -> byte[]
  * byte[] -> String
  * byte -> char
